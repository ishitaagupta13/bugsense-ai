# BugSense AI 🔍

> An AI-powered software analytics platform that predicts risky commits and bug-prone modules before failures occur.

## 🚀 Project Status
🟡 In Development — MVP Phase

## 🧱 Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | Java, Spring Boot, REST APIs |
| ML Service | Python, FastAPI, XGBoost, scikit-learn |
| Frontend | React.js |
| Database | PostgreSQL |
| DevOps | Docker, GitHub Actions |

## 📁 Project Structure
- `backend/` — Spring Boot REST API
- `ml-service/` — Python ML prediction service
- `frontend/` — React dashboard
- `docs/` — Architecture and documentation

## 📌 MVP Goals
- [ ] Connect a GitHub repository via API
- [ ] Fetch and store commit data
- [ ] Train a basic risk prediction model
- [ ] Serve predictions via REST API
- [ ] Display results on a dashboard
